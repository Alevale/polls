
# Demo

<a href="https://redux-polls.firebaseapp.com" target="redux-polls">got to app</a>

# To install
```bash
	npm install
```

# To lint
```bash
	npm run lint
```

# To test

```bash
	npm test
```

or

```bash
	npm run test:watch
```

# To dev start
```bash
	npm start
```

And goto [http://localhost:8080](http://localhost:8080)

# To dist
```bash
	npm run dist
```

# To deploy to Firebase
```bash
	npm run deploy
```
